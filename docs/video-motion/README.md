# PB Prono CDM - Pack video motion design

Objectif : produire une video courte qui explique le jeu PB Prono CDM, montre les ecrans principaux, et met en avant le premier prix.

Format recommande :
- Principal : vertical 9:16, 1080 x 1920, 45 a 60 secondes.
- Variante utile : carre 1080 x 1080 pour feed Instagram/Facebook.
- Sous-titres : obligatoires, beaucoup de vues se font sans son.

Fichiers de ce dossier :
- `storyboard.md` : scene par scene avec visuel, animation, texte ecran et voix-off.
- `voiceover-elevenlabs.txt` : script propre pour ElevenLabs.
- `screen-board.html` : planche d'ecrans mockup mobile a capturer/exporter.

## Methode recommandee

Pour aller vite et avoir un rendu propre :

1. Faire la voix dans ElevenLabs avec le script `voiceover-elevenlabs.txt`.
2. Exporter les ecrans depuis `screen-board.html` en captures PNG.
3. Monter la video dans CapCut, Canva, Premiere ou After Effects.
4. Utiliser Rive uniquement pour les animations vectorielles : logo PB, score qui monte, carte match, classement, confettis, fleches, transitions.

Rive est tres bien si tu veux une animation premium, mais pour une video client simple, le workflow le plus efficace est :

ElevenLabs pour la voix + CapCut/After Effects pour le montage + Rive pour quelques assets animes.
