# Storyboard - Video PB Prono CDM

Format : 9:16, 1080 x 1920.
Duree cible : 50 secondes.

## Scene 1 - Hook

Duree : 0:00 - 0:04

Visuel :
- Logo PB au centre.
- Fond sombre avec motif PB.
- Ballon / score / drapeaux en mouvement.

Texte ecran :
`PRONOSTIQUE LA COUPE DU MONDE 2026`

Voix-off :
`La Coupe du Monde arrive. Chez PB, tu ne fais pas que regarder les matchs. Tu joues.`

Animation :
- Logo qui apparait avec un leger zoom.
- Texte qui claque en deux lignes.

## Scene 2 - Connexion

Duree : 0:04 - 0:09

Visuel :
- Ecran login / creation pseudo.
- Boutons Google et Facebook.

Texte ecran :
`CONNECTE-TOI`
`CHOISIS TON PSEUDO`

Voix-off :
`Connecte-toi gratuitement, choisis ton pseudo, et entre dans le classement.`

Animation :
- Le telephone glisse depuis le bas.
- Un curseur tape un pseudo.

## Scene 3 - Calendrier

Duree : 0:09 - 0:15

Visuel :
- Liste des matchs.
- Carte France vs Maroc ou Senegal vs Bresil.
- Bouton pronostiquer.

Texte ecran :
`CHOISIS UN MATCH`

Voix-off :
`Avant chaque match, choisis la rencontre que tu veux pronostiquer.`

Animation :
- Scroll vertical leger.
- Une carte match se met en avant.

## Scene 4 - Pronostic score

Duree : 0:15 - 0:23

Visuel :
- Ecran pronostic avec deux scores.
- Boutons plus / moins.
- Score 2 - 1.

Texte ecran :
`ENTRE TON SCORE`
`AVANT LE COUP D'ENVOI`

Voix-off :
`Entre le score final avant le coup d'envoi. Une fois le match lance, ton prono est verrouille.`

Animation :
- Les scores passent de 0-0 a 2-1.
- Petit cadenas qui apparait.

## Scene 5 - Points

Duree : 0:23 - 0:31

Visuel :
- Carte score exact.
- Carte bon resultat.
- Points qui s'ajoutent.

Texte ecran :
`SCORE EXACT = 7 POINTS`
`BON RESULTAT = 3 POINTS`

Voix-off :
`Score exact : sept points. Bon resultat : trois points. Plus tu es precis, plus tu montes.`

Animation :
- Compteur de points qui augmente.
- Badge `+7 pts`.

## Scene 6 - Classement

Duree : 0:31 - 0:39

Visuel :
- Ecran classement.
- Pseudo utilisateur qui monte dans le top.

Texte ecran :
`MONTE AU CLASSEMENT`

Voix-off :
`Suis ton classement en direct et tente de finir tout en haut.`

Animation :
- Ligne utilisateur qui remonte de la 8e a la 1re place.
- Halo or autour de la premiere place.

## Scene 7 - Premier prix

Duree : 0:39 - 0:47

Visuel :
- Fond premium.
- Carte `1er prix`.
- Photo / illustration Maroc : riad, desert, Marrakech, soleil.

Texte ecran :
`LE JOUEUR AVEC LE PLUS DE POINTS`
`GAGNE UN WEEK-END AU MAROC`

Voix-off :
`Le joueur avec le plus de points remporte le premier prix : un week-end au Maroc.`

Animation :
- Transition classement vers destination Maroc.
- Ticket voyage qui apparait.

## Scene 8 - Lots PB + CTA

Duree : 0:47 - 0:55

Visuel :
- Menu PB : plat + boisson.
- Logo PB.
- Bouton final.

Texte ecran :
`LES MEILLEURS GAGNENT AUSSI`
`UN MENU PB`
`PLAT AU CHOIX + BOISSON`

Voix-off :
`Et les meilleurs joueurs gagnent aussi un menu PB : plat au choix plus boisson.`

Animation :
- Menu qui apparait.
- Logo final.

## Scene 9 - Final

Duree : 0:55 - 1:00

Visuel :
- Logo PB Prono.
- URL ou QR code si disponible.

Texte ecran :
`JOUE GRATUITEMENT`
`PB PRONO CDM`

Voix-off :
`PB Prono CDM. Joue gratuitement, pronostique, et tente de gagner.`

Animation :
- Fade out propre.
- Musique finit sur un impact.
